5G QoS Identifier (5QI) reference table

Comprehensive list of standardized 5QI values with QoS characteristics

What is 5QI?

The 5G QoS Identifier (5QI) is a scalar value used in 5G networks to identify specific QoS (Quality of Service) forwarding behaviors. Each 5QI value maps to a set of QoS characteristics including resource type, priority level, packet delay budget, and packet error rate.

5QI values 1-127 are standardized by 3GPP, while values 128-254 are operator-specific. Value 255 is reserved.

GBR: Guaranteed Bit Rate (dedicated resources)
Non-GBR: Best effort, shared resources
Delay Critical GBR: Ultra-low latency guaranteed
